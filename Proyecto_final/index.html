Bienvenido a Mi Proyecto de 16 Semanas

Este proyecto documenta mi progreso y aprendizaje a lo largo de 16 semanas intensas de desarrollo y crecimiento profesional. Cada semana contiene información sobre lo que aprendí, los proyectos que realicé y los desafíos que superé.

¿De qué se trata?

Este sitio web es un portafolio interactivo donde puedes explorar las 16 semanas de mi experiencia. Cada semana tiene su propia página dedicada con detalles específicos sobre mi progreso.

Características

  • 16 páginas individuales de semanas
  • Navegación fácil y intuitiva
  • Diseño responsive para todos los dispositivos
  • Información detallada de cada semana
  • Sección "Acerca de mí" para conocer mi historia

Comenzar

Usa el menú de navegación superior para explorar las diferentes secciones. Haz clic en "Semanas" para ver todas las semanas o "Acerca de mí" para conocer más sobre mí.

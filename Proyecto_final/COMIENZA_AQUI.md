## 🚀 PRIMEROS 5 PASOS (10 minutos)

### 1️⃣ Clona tu repositorio
```bash
git clone https://github.com/josue12335/Proyecto_final.git
cd Proyecto_final
```

### 2️⃣ Crea las 16 carpetas de semanas
```bash
# Windows (PowerShell)
1..16 | ForEach-Object { mkdir "semana$_" }

# Mac/Linux
mkdir semana{1..16}
```

### 3️⃣ Sube todo a GitHub
```bash
git add .
git commit -m "Inicializar portafolio"
git push origin main
```

### 4️⃣ Activa GitHub Pages
1. Ve a: https://github.com/josue12335/Proyecto_final/settings/pages
2. Source: "Deploy from a branch"
3. Branch: `main` / `/ (root)`
4. Guarda

### 5️⃣ Espera 5 minutos y abre:
```
https://josue12335.github.io/Proyecto_final.html
```

✅ **¡Deberías ver tu portafolio!**

Una vez que tengas todo configurado, para cada semana:

